### Based on:
- Spring Boot
- Spring Web MVC
- Spring Data JPA ( Hibernate )
- Apache Shiro
- Java 8
- Maven

## Build and Run
1. Make sure JDK 8 and Maven are installed, and enter the root directory.
2. Build: `mvn clean package`
3. Run: `mvn spring-boot:run`
4. Visit the homepage: `http://localhost:8080`
